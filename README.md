# 📝 Lista de Tarefas

Projeto de lista de tarefas (To-Do List) desenvolvido com foco em prática de lógica de programação e manipulação do DOM.

---

## 🚀 Funcionalidades

- ✅ Adicionar novas tarefas  
- 🗑️ Remover tarefas  
- ✏️ Marcar tarefas como concluídas  

---

## 🛠️ Tecnologias utilizadas

- HTML  
- CSS  
- JavaScript
